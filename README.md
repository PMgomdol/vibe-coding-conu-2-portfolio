# 안재찬 포트폴리오

[danielsun.space](https://danielsun.space/) 디자인·인터랙션을 참고하여 제작한 개인 포트폴리오 사이트입니다.
HTML / CSS / JS 단일 파일로 구성되어 있습니다.

## 구성

| 섹션 | 내용 |
|---|---|
| Hero | 캔버스 선레이 애니메이션 + 대형 타이포그래피 |
| Work | 6개 프로젝트 비대칭 그리드 카드 |
| About | 소개글 + 숫자 통계 + 인용구 |
| Skills | 다크 테마 도구 카드 6종 |
| Process | 4단계 업무 방식 (노란 배경) |
| Contact | 이메일 CTA |

## 인터랙션

- 커스텀 커서 (점 + 링, 호버 확대)
- 히어로 선레이 Canvas 애니메이션 (자동 회전)
- 스크롤 parallax (히어로 이름)
- 스크롤 reveal 애니메이션 (IntersectionObserver)
- 키워드 자동 스크롤 ticker
- 플로팅 pill 네비게이션 + 활성 섹션 하이라이트
- 프로젝트 카드 호버 효과
- 페이지 로딩 인트로

## 실행

별도 빌드 없이 `index.html`을 브라우저에서 바로 열거나, 로컬 서버로 실행합니다.

```bash
npx serve .
```

## 배포

Netlify, GitHub Pages 등에 `index.html` 단일 파일 업로드로 바로 배포 가능합니다.
